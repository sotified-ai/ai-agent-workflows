Core System Architecture Specification

Solving the triad of Automated Test Generation, Long-term Context Retention, and Compliance-Safe Local Deployment through a config-driven Agent Factory.

1

Test Generation Quality

Flat chats hallucinate APIs, skip boundaries, and produce unstructured output. We resolve this with a structured, feedback-driven State Graph pipeline where a specialized audit agent reviews the suite and triggers loops for automatic self-correction.

2

Context Windows & Memory

Complex codebases and manuals overflow context windows, causing LLM amnesia. We resolve this via a central state graph managing a ledger of summaries, and query the vector store on-the-fly to pull only relevant context for the current test scope.

3

Compliance & Air-gapped Host

Corporate compliance prohibits sending passwords, schema details, or source code to public clouds. ComplianceGuard performs dual-way masking (replacing secrets with local reversible tokens) and facilitates 100% offline local inference.

State Graph & Proxy Lifecycle Map

[INPUT LAYER] Project Knowledge Base ➔ Requirements DB ➔ User Stories & Swagger Specifications
[COMPLIANCE SECURITY] ComplianceGuard Scrubbing Proxy ➔ Local Cache (Prevents repetitive LLM costs)
[FACTORY LAYER] AgentFactory loads JSON definitions ➔ Dynamically instantiates DynamicAgents
[FLOW ENGINE] Orchestrator State-Graph DAG (Orchestrates transitions & self-repair loops)
[OUTPUT FORMAT] Traceability Validator ➔ Styled Multi-Sheet Excel Workbook Export
